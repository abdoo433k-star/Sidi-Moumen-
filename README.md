# Sidi-Moumen-

مشروع تطويري متكامل يتضمن صفحة ويب مع أنماط وتفاعلات.

## البنية الأساسية

```
Sidi-Moumen/
├── public/
│   ├── css/
│   │   └── style.css      # ملف الأنماط
│   ├── js/
│   │   └── app.js         # ملف السكريبت
│   └── index.html         # الصفحة الرئيسية
├── README.md
└── package.json
```

## المتطلبات

- متصفح ويب حديث
- محرر نصوص (VS Code, Sublime Text, إلخ)

## البدء

1. استنسخ المستودع
2. افتح ملف `public/index.html` في متصفحك
3. ابدأ بتطوير المشروع
